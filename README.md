# Tic-Tac-Toe-AI
Using Minimax, implement an AI to play Tic-Tac-Toe optimally.

Visit [Tic-Tac-Toe-CS50's AI](https://cs50.harvard.edu/ai/2020/projects/0/tictactoe/) for more information.
